    使用hugo+GitHub Page搭建个人网站

    一/软件安装准备

    1. hugo—
    2. GitHub客户端—网页代码托管
    3. Typora—

    一、hugo安装及配置

    1. 在hugo的官方下载地址下载对应的操作系统版本安装文件，以windows64位操作系统为例，下载对应的hugo_0.81.0_Windows-64bit.zip。

    2. 下载完成后，解压得到以下文件。

    3. 配置环境变量。

      1. 如步骤2所示，hugo.exe当前路径为F:\hugo\bin。

      2. 右键点击此电脑，选择”属性“，打开控制面板后选择”高级系统设置“。

      3. 选择环境变量，配置系统变量path。

      4. 填入hugo.exe所在的路径，如图，当前安装路径为F:\hugo\bin。配置完成后点击“确定”。

    4. 打开命令行，执行以下命令查看是否安装成功。

      hugo version

      hugo v0.81.0-59D15C97 windows/amd64 BuildDate=2021-02-19T17:07:12Z VendorInfo=gohugoio 显示该条即表示安装成功。

    5. 进入F:\hugo下，打开命令窗口，执行如下命令。

      hugo new site文件名称（如blog）

      执行后，在hugo目录下会生成一个blog文件夹

    6. 进入blog目录下，执行以下命令。

      hugo new post/test.md

      执行后，在content/post目录下会生成test.md文件，这个文件就是网页的主要内容，需要用markdown编辑工具，如Typora来编辑内容。同时需要将文件头部的draft改为false，这样部署了才能看到文章。

    7. 进入blog下的themes目录。

      cd themes

    8. 执行以下命令下载主题。

      git clone https://github.com/EmielH/tale-hugo.git

      （可在官网选择想要的主题下载）

    9. 到blog目录下打开config.toml配置指定主题。如theme = “tale-hugo” 。配置完成后执行如下命令：

      hugo server

      执行后显示如下：

      可访问：http://localhost:1313/ 查看效果。

    二、Github Page配置

    1. 使用自己的账号登录Github，登录后，点击右上角，出现下拉菜单，点击 Your repositories 进入页面。
    2. 点击 New，新建。
    3. 进入 Creat a new repository 页面，需要将yourname换成自己GitHub的用户名，即图中owner显示的用户名。最后点击create repository即创见完毕。
    4. 在GitHub Pages官网https://pages.github.com/下载GitHub客户端，安装后并登录。
    5. 在客户端clone刚创建的仓库到本地。

    三/Hugo关联刚创建的仓库

    • 在hugo站点目录config.toml中baseURL要换成自己建立的仓库，如baseURL =”https://905497297.github.io/“

    • 进入站点根目录下，执行:

      $ hugo

      执行后，站点根目录下会生成一个 public 文件夹，该文件下的内容即Hugo生成的整个静态网站。

    四/将网站push至GitHub

    • 打开GitHub本地文件夹
    • 放入自己的网站文件
    • 回到客户端
    • push上去
    • 上传完后回到网页版，发现已经上传了相关的文件
    • 在setting里找到博客的链接地址
    • 该链接即为网站的地址：
